# 🎓 College Chatbot System


## 📖 Description

The College Chatbot System is a web-based application designed to help students get instant information about college-related queries such as notices, timetable, events, and general FAQs.

It also provides an Admin Panel where administrators can manage student data, notices, events, and system content efficiently.

## 🚀 Features

### 👨‍🎓 Student
- Student Login Authentication
- Access Chatbot for queries
- View Dashboard with personal details
- Check Notices, Timetable, Events
- Logout functionality

### 🛠 Admin
- Admin Login
- Manage Students
- Add / Update Notices
- Manage Timetable
- Manage Events
- Monitor system

### 🤖 Chatbot
- Answer student queries
- Keyword-based response system
- Fast and interactive UI


## 💻 Technologies Used

### Frontend
- React.js
- HTML5, CSS3
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Tools
- VS Code
- Postman

## ⚙️ Installation

1. Clone the repository:
   git clone https://github.com/your-username/college-chatbot.git

2. Install frontend dependencies:
   cd frontend
   npm install

3. Install backend dependencies:
   cd backend
   npm install

4. Start backend server:
   node index.js

5. Start frontend:
   npm run dev


## 🔄 Working Process

### Student Flow
1. Login
2. Access Dashboard
3. Open Chatbot
4. Ask Query
5. Get Response

### Admin Flow
1. Login
2. Access Dashboard
3. Manage Data (Students, Notices, Events)
4. Monitor System
